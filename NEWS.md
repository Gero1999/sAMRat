# sAMRat 0.1.0

## New Features

* Initial release of sAMRat - Shiny Antimicrobial Resistance Analysis Tool
* Interactive Shiny application for AMR data analysis
* Core function `create_amr_obj()` for creating AMR-compatible data objects
* Data upload and mapping interface
* Modular Shiny application structure with reusable components
* Basic AMR analysis and visualization capabilities

## Documentation

* Added comprehensive pkgdown website with:
  - Getting Started guide
  - Architecture documentation
  - Contributing guidelines
* Package documentation using roxygen2
* Code of Conduct and Contributing guidelines

## Infrastructure

* GitHub Actions workflow for pkgdown website deployment
* Basic test infrastructure using testthat
* Linting configuration with lintr
